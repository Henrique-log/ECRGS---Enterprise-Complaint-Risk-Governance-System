# ECRGS Backend – FastAPI MVP

This is a minimal FastAPI implementation of the ECRGS governance core.

## Run Locally

1. Navigate to the backend folder:

cd backend

2. Install dependencies:

pip install -r requirements.txt

3. Start the server:

uvicorn main:app --reload

## API Documentation

After running, access:

http://127.0.0.1:8000/docs

This exposes the interactive Swagger UI.
